просто vuejs базирано куиз spa приложениe

#download
```
git clone https://github.com/naelir/cq_spa_demo.git
```

#билд на клиента
```
cd cq_spa_demo\src\main\resources
npm install
npm run build
```
#билд на сървъра и старт
```
cd cq_spa_demo
mvn spring-boot:run
http://localhost:8080
```
